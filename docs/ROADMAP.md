# Roadmap

## Version 1

- Static GitHub Pages deployment
- Installable PWA
- Offline application shell
- U.S., Canada, Mexico, territories, and selected special categories
- Bundled simplified jurisdiction boundaries
- Current-position marker
- Automatic current-jurisdiction detection
- Manual location correction
- Plate map and searchable picker
- Original plate illustrations
- One collection per jurisdiction per trip
- Local distance calculations
- Cooperative and competitive local play
- Achievements
- Collection book
- Trip timeline and summary
- JSON import and export
- No backend dependency

## Version 1.1

- Expanded plate artwork and historical design variants
- Improved route visualization
- Printable trip report
- Shareable summary image
- Additional accessibility testing
- More achievements and regional challenges

## Version 2

Optional online services that do not replace local-first play:

- Accounts
- Encrypted cloud backup
- Multi-device trip synchronization
- Family invitations
- Shared live trips
- Community rarity statistics
- Seasonal challenges
- Plate-data update service

Version 2 synchronization must preserve offline-first behavior. The local database remains authoritative while disconnected.
